### Mock Creation Helpers

```python
def create_mock_api_client(base_url="https://api.example.com"):
    """
    Create a configured mock API client.

    Args:
        base_url: Base URL for the API

    Returns:
        Mock object configured as API client
    """
    mock_client = Mock()
    mock_client.base_url = base_url
    mock_client.get = Mock(return_value={"status": "ok"})
    mock_client.post = Mock(return_value={"id": 1})
    return mock_client


def create_mock_database():
    """
    Create a configured mock database connection.

    Returns:
        Mock object configured as database connection
    """
    mock_db = Mock()
    mock_db.connect = Mock(return_value=True)
    mock_db.disconnect = Mock(return_value=True)
    mock_db.query = Mock(return_value=[])
    return mock_db
```

### Test Data Builders

```python
def build_test_user(user_id=1, name="Test User", email="test@example.com", **kwargs):
    """
    Build a test user object with default values.

    Args:
        user_id: User ID
        name: User name
        email: User email
        **kwargs: Additional user attributes

    Returns:
        User object or dict with user data
    """
    user_data = {
        "id": user_id,
        "name": name,
        "email": email,
        "age": kwargs.get("age", 30),
        "role": kwargs.get("role", "user")
    }
    user_data.update(kwargs)
    return user_data


def build_test_request(method="GET", path="/", body=None, headers=None):
    """
    Build a test HTTP request object.

    Args:
        method: HTTP method
        path: Request path
        body: Request body
        headers: Request headers

    Returns:
        Mock request object
    """
    request = Mock()
    request.method = method
    request.path = path
    request.body = body or {}
    request.headers = headers or {}
    return request
```

### Setup/Teardown Utilities

```python
@pytest.fixture
def test_database():
    """
    Pytest fixture for database setup and teardown.

    Yields:
        Database connection

    Cleanup:
        Closes database connection after test
    """
    from database import Database
    db = Database("test.db")
    db.connect()
    yield db
    db.disconnect()


@pytest.fixture
def temp_directory(tmp_path):
    """
    Pytest fixture for temporary directory setup.

    Args:
        tmp_path: pytest built-in tmp_path fixture

    Yields:
        Path to temporary directory

    Cleanup:
        Removes temporary files after test
    """
    test_dir = tmp_path / "test_data"
    test_dir.mkdir()
    yield test_dir
    # Cleanup handled automatically by tmp_path
```

## Customization

Generated helpers can be customized by users:

```python
def create_mock_api_client(base_url="https://api.example.com", timeout=30):
    """Extended with timeout parameter."""
    mock_client = Mock()
    mock_client.base_url = base_url
    mock_client.timeout = timeout
    mock_client.get = Mock(return_value={"status": "ok"})
    return mock_client
```

## Best Practices

2. **Docstrings**: Keep docstrings clear and concise
3. **Defaults**: Provide sensible defaults for all parameters
4. **Flexibility**: Allow overriding via `**kwargs`
5. **Documentation**: Document return types and side effects
